A Proxy Certificate allows you to browse the web without continuous certificate errors. Pressing Install will download the latest certificate and launch the Certificate Manager.

Once it launches, you'll need to press and complete the installation manually.
Return to this app to enable the proxy, then be sure you reboot your device before browsing the web.

This app provides defaults for connecting to the webOS Archive proxy service, but can be used with any proxy. To learn more about the webOS Archive, or to sign up for the (free!) proxy service, visit www.webosarchive.org.
Based on Proxy Set Basic by VerusOra. Modifications by webOS Archive, 2022.
